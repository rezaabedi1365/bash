
# Text seperator command 

   * [awk](####rd)
     for seperate coloumn
     
   * [sed](####find)
     for seperate raw
     
   * [cut](####XCOPY)
     for seperate charachter with devision
     
   * [ grep ](####robocopy)
     for select line with wildcard
     
   * [ head ](####rd&rmdir)
     for select first or lasts line
